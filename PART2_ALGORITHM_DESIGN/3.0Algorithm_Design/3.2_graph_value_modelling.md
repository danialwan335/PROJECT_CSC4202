# 3.2 Graph Value Modelling

## Road Network Graph Model

The post-landslide road network is modelled as a weighted undirected graph. Edge weights represent the estimated ambulance travel time (in minutes) on each surviving road segment.

### Adjacency List Representation

The graph is stored as an adjacency list: each node stores a list of (neighbour, weight) pairs. This is space-efficient for sparse graphs (few surviving roads after a landslide) and allows O(1) edge access per neighbour during Dijkstra's relaxation.

### Optimal Path (Traced by Dijkstra's)

The shortest path from Hospital (0) to Kampung Sungai Lui (7):

**0 → 1 → 2 → 4 → 6 → 7**

| Segment | Travel Time |
|---|---|
| 0 → 1 | 8 min |
| 1 → 2 | 10 min |
| 2 → 4 | 20 min |
| 4 → 6 | 10 min |
| 6 → 7 | 14 min |
| **Total** | **62 min** |

The alternative routes via Sungai Chilling and Pos Gertak yield longer total times, confirming the optimality of the above path.
