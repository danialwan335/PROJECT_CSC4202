## Function: BuildGraph

```
Function BuildGraph(edgeList, numVertices):
    Create adjacency list adj[0..numVertices-1] — each entry is an empty list
    For each edge (u, v, w) in edgeList:
        Append (v, w) to adj[u]
        Append (u, w) to adj[v]        // undirected: add both directions
    Return adj
```

## Function: Dijkstra

```
Function Dijkstra(adj, V, source):
    // Initialisation
    dist[0..V-1] ← ∞
    prev[0..V-1] ← -1
    visited[0..V-1] ← false
    dist[source] ← 0

    PQ ← MinPriorityQueue()
    PQ.insert(0, source)               // (distance, nodeId)

    // Main relaxation loop
    While PQ is not empty:
        (d, u) ← PQ.extractMin()

        If visited[u]:                 // stale entry — skip
            continue

        visited[u] ← true

        For each (v, w) in adj[u]:    // relax all edges from u
            If NOT visited[v] AND dist[u] + w < dist[v]:
                dist[v] ← dist[u] + w
                prev[v] ← u
                PQ.insert(dist[v], v)

    Return dist, prev
```

## Function: ReconstructPath

```
Function ReconstructPath(prev, destination):
    path ← empty list
    current ← destination

    While current ≠ -1:
        path.prepend(current)
        current ← prev[current]

    If path[0] ≠ source:              // destination was unreachable
        Return "No path exists"

    Return path
```
